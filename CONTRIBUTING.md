# Contributing

Thank you for your interest in CGAO.

At the current stage, this project is under active development.

Contributions are welcome after v1.0.0.

## Development Workflow

1. Create a new branch.

```
git checkout -b feature/your-feature
```

2. Implement your feature.

3. Run tests.

4. Submit a Pull Request.

---

## Code Style

- Python 3.13+
- Black formatting
- Type hints
- Clear docstrings

---

## Reporting Issues

Please open a GitHub Issue describing:

- Environment
- Reproduction steps
- Expected behavior
- Actual behavior
